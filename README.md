# Workflow for Posting

## Create a file in '\_posts'
Follow the naming scheme as defined here: '2018-08-20-bananas.md', ie. 'date-name.md'.
The header portion of the Markdown file must be notated as follows:
```
---
layout: post
author: jill
---
```
Determining the layout (ie. formatting template) to follow and any other information necessary to fill out that template, for example: the author of the post.
