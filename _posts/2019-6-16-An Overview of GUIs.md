The <strong>Graphical User Interface</strong> is a programmer's foward facing representative. Each platform has a different design philosophy based on the language of choice and intended user. The frontend medias that are popular today are <i> Websites </i>, <i> Mobile Applications </i>, and <i> Desktop Applications </i>. 

### Websites
I will keep this fairly brief. HTML, CSS, and Javascript are the keystones of all websites. HTML is the organization of the page. HTML is a spin off of XML which is a more generalizable Markup Language. XML is used in Maven .pom files for those who are familiar with Java build tools. CSS stylizes yoru website, adding visual panache to your elements. You can change colors, shift around elements, and even do animations in CSS.
